# Marpogi — Rapper Portfolio WordPress Theme

A dark, urban-themed WordPress theme designed for rapper **Marpogi**. Features SoundCloud integration, bold typography, animated UI, and a hip-hop inspired aesthetic.

![Theme Preview](screenshot.png)

## Features

- **Dark Urban Design** — Deep black backgrounds with cyan accent (#00b9f0)
- **SoundCloud Integration** — Embedded SoundCloud player with your "I Luv U" playlist
- **Responsive** — Fully mobile-friendly with hamburger menu
- **Animated UI** — Glitch text effects, floating particles, equalizer bars, scroll animations
- **WordPress Customizer** — Edit hero text, about section, and social links from the dashboard
- **Multiple Templates** — Front page, blog, single post, and page templates

## Installation

1. Download the `marpogi-rapper-theme.zip` file
2. Go to your WordPress Dashboard → **Appearance** → **Themes** → **Add New** → **Upload Theme**
3. Select the zip file and click **Install Now**
4. Click **Activate**

## Customization

Go to **Appearance** → **Customize** to edit:

- **Hero Section** — Subtitle and tagline text
- **About Section** — Your bio text
- **Social Links** — SoundCloud, Instagram, YouTube, Spotify URLs, and contact email

## Sections

| Section  | Description                                        |
| -------- | -------------------------------------------------- |
| Hero     | Full-screen intro with glitch title and CTA buttons |
| Music    | SoundCloud embed with equalizer animation           |
| About    | Bio, genre tags, and profile image placeholder      |
| Visuals  | Gallery grid with hover overlays                    |
| Contact  | Social links and email                              |

## Standalone Preview

Open `preview.html` in any browser to see the theme without WordPress.
